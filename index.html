🎵 Multi-Playlist Pro

Guía de uso e información

Esta aplicación te permite gestionar una lista de reproducción unificada desde distintas fuentes.

  • Títulos: Debes escribir el nombre de la canción manualmente en el campo "Nombre" antes de añadir el enlace.
  • YouTube / Spotify / SoundCloud: Pega el enlace de compartir directamente.
  • Google Drive / Dropbox: El archivo debe estar configurado como Público ("Cualquier persona con el enlace").
  • Archivos Locales: Usa el botón 📁 para cargar MP3. Estos se guardan en la base de datos interna de tu navegador (IndexedDB).

* Nota: Para pasar la lista a otro dispositivo, usa el botón "Exportar JSON" y cárgalo en el nuevo terminal. Los archivos locales (📁) no se transfieren por JSON.
